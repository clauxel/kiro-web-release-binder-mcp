# Workflow

Kiro Web Release Binder is a paid remote MCP release gate for Kiro projects, release goals, acceptance criteria, owner signoff, evidence binders, and CI-friendly approval receipts.

## Repeatable Flow

1. Submit the Kiro project, repository, release goal, and acceptance criteria.
2. Check missing evidence, environment gaps, and approval status.
3. Return allow, review, or deny for CI and agent workflows.
4. Export a release binder for owners and clients.

## Output Mindset

The useful artifact is not a marketing claim. It is evidence that a reviewer can inspect, export, and compare later.
